# biglinux-autotools
Scripts em Python para automação de pós-instalação e manutenção diária no BigLinux. O setup.py instala programas essenciais (Browsers, Office, Dev) e configura o ZapZap em PT-BR. O atualizar_sistema.py atualiza repositórios, AUR, Flatpaks e realiza uma limpeza profunda de caches, logs e órfãos.
